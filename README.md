# FlightBookingSystem

## Goal (v0.1)
- Reactive Spring Boot application (WebFlux) with MongoDB.
- Support user search, booking, ticket retrieval, booking history, and cancellation.
- REST endpoints under `/api/v1.0/flight/...`
